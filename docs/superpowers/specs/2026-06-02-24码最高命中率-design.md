# 24码最高命中率方案设计

**日期**: 2026-06-02
**目标**: 49选24，找出历史最高命中率方案

---

## 背景数据

- 总期数：1437期（2024-2026）
- 后区20-49基础命中率：61.75%
- 前区高频号码：01、06、10、16、23

---

## 架构

```
输入: history (历史记录)
      strategy = "freq_only" | "gap_freq" | "multi_window"
      top_k = 24

流程:
1. 解析特码号码频率和遗漏值
2. 按策略计算综合分数
3. 排序取top_k
4. 输出号码列表
```

---

## 策略设计

### 方案甲：全频率扫描 (freq_only)
- `score = frequency`
- 按频率降序取top_k
- 参数：lookback = 50/100/全部

### 方案乙：遗漏值×频率 (gap_freq)
- `score = frequency × gap^weight`
- 参数：lookback, gap_weight ∈ [0.3, 0.5, 0.7, 1.0]
- 当前v41已验证此策略

### 方案丙：滚动窗口多策略 (multi_window)
- 近20期、50期、100期分别计算频率
- mode: union（并集）或 intersection（交集）
- 参数：windows, mode
